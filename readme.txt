This widget shows posts that are stickied under a certain category.

== Widget options ==
- Widget title
- Category (select from a drop-down)
- Max posts to show
- Show title only - whether to show the full post or just the linked title

== Installation ==

1. Upload `sticky-cat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I use this? =

Go to the Appearance menu, and open the Widgets submenu. There you will find the widget under the title 'ShowMeTheStickiesInTheCat'.
